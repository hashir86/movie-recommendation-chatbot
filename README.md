# 🎬 AI Movie Recommendation Chatbot

An intelligent entertainment chatbot built using Streamlit, TMDB API, and Google Generative AI.  
It recommends movies based on genres, actors, directors, or similar titles, and also supports general conversation using AI.

---

## 🚀 Features

- 🎥 Movie recommendations by:
  - Genre (e.g., action, comedy)
  - Actor (e.g., Brad Pitt)
  - Director
  - Similar movies
- AI-powered chat using Google Gemini
- Multi-session chat system
- Context-aware conversation (remembers history)
- Randomized recommendations for variety
- Session saving and loading

---

## 🛠️ Tech Stack

- Python
- Streamlit
- TMDB API (The Movie Database)
- Google Generative AI (Gemini)
- Requests
- Regex (for intent detection)
